# Java-Payment-System-OOP

This project was developed for the **COMP2310 Java Programming course** at **Birzeit University**.

## Project Description

The system implements a payment processing system using Object-Oriented Programming concepts in Java.

The system includes the following classes:

- CustomerPayment
- Cash
- Check
- CreditCard
- Payable Interface

Each payment type calculates its payment differently:

- Cash → applies discount
- Check → payment equals amount
- CreditCard → adds charging fee
